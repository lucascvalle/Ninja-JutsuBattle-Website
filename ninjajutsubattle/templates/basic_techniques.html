Basic Techniques

Every Jutsu that doesn't belong to an Element or a Kekkei Genkai is a Basic Technique.

Every ninja that graduated from the Ninja Academy knows how to perform 3 Basic Techniques of Rank C, and a ninja can have up to 11 Basic Techniques, as follows:

  • 5 Rank C
  • 3 Rank B
  • 2 Rank A
  • 1 Rank S
